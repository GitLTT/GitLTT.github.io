jQuery 中的遍历：

  • jQuery 遍历 ：用于根据其相对于其他元素的关系来“查找”（或选取）HTML 元素。

    以某项选择开始，并沿着这个选择移动，直到抵达期望的元素为止。

  • jQ遍历分为四种，1. 想上遍历 2. 向下遍历 3. 过滤 4. 水平遍历

  1. .eq 过滤遍历

    • 用法

      $(‘li’).eq(0).css(‘background’,’red’);

    • 说明：

      查找获取到所有li标签中的第几个，从零开始

  2. .first() 过滤遍历

    • 用法

      $(‘li’).first().css(‘background’,’red’);

    • 说明：

      查找获取到所有li标签中的第一个

  3. .last() 过滤遍历

    • 用法

      $(‘li’).last().css(‘background’,’red’);

    • 说明：

      查找获取到所有li标签中的最后一个

  4. .slice() 过滤遍历

    • 用法

      $(‘li’).slice(2,4).css(‘background’,’red’);

    • 说明：

      查找获取到所有li标签中的第几个到第几个，下标从零开始，要头不要尾

  5. .children(选择器) 向下遍历

    • 用法

      $(‘ul’).children(‘div’).css(‘background’,’red’);

    • 说明：

      查找获取到所有ul标签中的所有【直接子元素】是选择器的元素

  6. .find(选择器) 向下遍历

    • 用法

      $(‘ul’).find(‘div’).css(‘background’,’red’);

    • 说明：

      查找获取到所有ul标签中的所有后代是选择器的元素

  7. .parent() 向上遍历

    • 用法

      $(‘ul’).parent().css(‘background’,’red’);

    • 说明：

      返回ul的直接父元素

  8. 更多遍历方法

    • 向上遍历 （查找元素祖先）

      • parent() 返回被选元素的直接父元素 (只会向下一级对 DOM 树进行遍历 )

        例：

        $(“span”).parent();

      • parents() 返回被选元素的所有祖先元素，它一路向上直到文档的根元素 ()

        例：

        $(“span”).parents();$(“span”).parents(“ul”)

        返回所有 元素的所有祖先，并且它是

          元素

      • parentsUntil() 返回介于两个给定元素之间的所有祖先元素

        例：

        $(“span”).parentsUntil(“div”);

    • 向下遍历（查找元素后代）

      • children() 返回被选元素的所有直接子元素(只会向下一级对 DOM 树进行遍历 )

        例：

        $(“div”).children()$(“div”).children(“p.1”)

        返回类名为 “1” 的所有

        元素，并且它们是

        的直接子元素

      • find() 返回被选元素的后代元素，一路向下直到最后一个后代

        例：

        $(“div”).find(“span”);$(“div”).find(“*”);返回

        的所有后代

    • 水平遍历方法(元素同胞元素)：

      • siblings() 返回被选元素的所有同胞元素

        例：

        $(“h2”).siblings();$(“h2”).siblings(“p”);

        返回属于

        的同胞元素的所有

        元素

      • next() 、prev() 返回被选元素的下一个/上一个同胞元素

        例：

        $(“h2”).next();

      • nextAll()、prevAll() 返回被选元素的所有跟随的同胞（前面同胞）元素

        例：

        $(“h2”).nextAll();

      • nextUntil()、prevUntil() 返回介于两个给定参数之间的所有跟随的同胞（前面同胞）元素

        例：

        $(“h2”).nextUntil(“h6”);

    • 过滤：

      • first() 返回被选元素的首个元素

        例：

        $(“div p”).first();

      • last() 返回被选元素的最后一个元素

        例：

        $(“div p”).last();

      • eq() 返回被选元素中带有指定索引号的元素

        例：

        $(“p”).eq(1);

      • filter() 不匹配这个标准的元素会被从集合中删除，匹配的元素会被返回

        例：

        $(“p”).filter(“.intro”);

      • not() not() 方法返回不匹配标准的所有元素not() 方法不 filter() 相反。

        例：

        $(“p”).not(“.intro”);
