this/字符串/Date

改变this指向：

1. call();

    a. 语法：call(thisObj，Object) 
    b. 定义：调用一个对象的一个方法，以另一个对象替换当前对象  

2. apply(); 

    a. 语法：apply(thisObj，[argArray]) 
    b.  定义：应用某一对象的一个方法，用另一个对象替换当前对象 


例：

3. call();和apply();的区别： 

    作用：都是修改this的指向； 

    参数：call(); 
        第一个：你要把this修改成那个对象； 
        第二个：从第二个参数开始每一个代表你要传递的实际参数 

          apply(); 
        第一个和call();相同 
        第二个：必须是一个数组，数组中的每一项代表你要传的值

字符串的操作：

1. 创建子字符串： 

    a. var a = "";     //typeof  :   string; 

    b. var a = new String("");     // typeof    :    object 

2. 属性：length (长度) 

3. 方法： 

    a. charAt();返回指定位置的字符 

        str.charAt(1); === str[0]; (但是不建议) 

    b. charCodeAt(); 返回指定位置的字符的Unicode编码 

    c. search() 用于检索字符串中指定的子字符串 

                查找的时候是按照查找第一个位置来看的， 

                 查找到后返回第一个字符的位置，差找不到后返回-1 

    d. replace() 查找匹配指定的字符串，然后用新字符串代替匹配的字符串 

        例：var a = str.replace("a","b"); 

            var b = str.replace(/a/g,"b");   // 进行全局更改 

    e. concat() 将两个或多个字符的文本组合起来，返回一个新的字符串 

        例：str.concat(str1);  // str+str1; 

    f. indexOf() 返回字符串中一个子串第一处出现的索引。如果没有匹配项，返回 -1 

    g. lastIndexOf() 返回字符串中一个子串最后一处出现的索引，如果没有匹配项，返回 -1 

    h. slice() 提取字符串的一部分，并返回一个新字符串。 

            两个参数，第一个为起始位置，第二个为终    止位置（留头不留尾），如果没有end，就取到末尾 

    i. toLowerCase() 将整个字符串转成小写字母 

    j. toUpperCase() 将整个字符串转成大写字母 

    k. 转义字符：转义字符 \ 

            我们平时想输出一个双引号（“”）我们会这样写： 

            document.write(“””); 

            但是这样写是错诨的,所以我们需要用到转义字符： 

            document.write(“ \””);

Date对象：

var date = new Date();

  • 获取时间的方法：

    1. Date()返回当日的日期和时间；

    2. getDate()从 Date 对象返回一个月中的某一天 (1 ~ 31)；

    3. getDay()从 Date 对象返回一周中的某一天 (0 ~ 6)；

    4. getMonth()从 Date 对象返回月份 (0 ~ 11)；

    5. getFullYear()从 Date 对象以四位数字返回年份；

    6. getHours()返回 Date 对象的小时 (0 ~ 23)；

    7. getMinutes()返回 Date 对象的分钟 (0 ~ 59)；

    8. getSeconds()返回 Date 对象的秒数 (0 ~ 59)；

    9. getMilliseconds()返回 Date 对象的毫秒(0 ~ 999)；

    10. getTime()返回 1970 年 1 月 1 日至今的毫秒数；

  • 设置时间方法：

    1. setHours()设置 Date 对象中的小时 (0 ~ 23)；

    2. setMinutes()设置 Date 对象中的分钟 (0 ~ 59)；

    3. setSeconds()设置 Date 对象中的秒钟 (0 ~ 59)；

    4. setMilliseconds()设置 Date 对象中的毫秒 (0 ~ 999)；

    5. setTime()以毫秒设置 Date 对象；
