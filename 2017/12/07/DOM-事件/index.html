DOM

通过选择器来获取节点：document.querySelectorAll();

获取属性：

创建节点：（从后进行添加）

节点操作：

节点查找：

获取下级节点：

1. children：子节点，不包含空节点

2. childNodes：子节点，包含空节点

3. firstChild：第一个子节点，包含空节点

4. firstElementChild：第一个子节点，不包含空节点

5. lastChild：最后一个子节点，包含空节点

6. lastElementChild：最后一个子节点，不包含空节点

获取兄弟节点：

1. nextSibling：下一个兄弟节点，包含空节点

2. nextElementSibling：下一个兄弟节点，不包含空节点

3. previousSibling：前一个兄弟节点，包含空节点

4. previousElementSibling：前一个兄弟节点，不包含空节点

获取父级节点：

1. parentNode：父节点

2. offsetParent：第一个有定位属性的父节点，如果没有，则返回body

事件：

普通事件：a.onclick = function(){};

    同一个元素，同一个事件驱动只能绑定一个，如果绑定两个，下面的事件会替换上面的；

1. onload 图片或页面加载完成

2. 焦点事件:onfocus/onblur 获得焦点/失去焦点

3. onchange 表单内容发送改变

4. onclick 点击

5. ondblclick 点击两次

6. onkeydown 键盘按下

7. onkeyup 键盘抬起

获取对象 、 兼容处理：

键盘event对象属性 keyCode

1. onmousedown 鼠标按下

2. onmouseup 鼠标抬起

3. onmousemove 鼠标移动

4. onmouseover 移到对象上

5. onmouseout 鼠标离开

6. 鼠标event对象属性 clientX clientY

7. onselect 在文本框中的文本被选中时发生,支持<input type="text">, <textarea>

8. onsubmit 在表单中的提交按钮被点击时触发

9. onreset 在表单中的重置按钮被点击时触发

10. onerror 在文档或图像加载过程中发生错误时被触发

11. 阻止默认事件preventDefault() 或者 return false；
