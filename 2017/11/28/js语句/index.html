数组赋值：

数组有一个length属性，记录我们数组中元素的个数/数组长度；

数组的方法：重点

1.join(); 

    把数组转换成字符串；得到一个新的字符串，不会修改我们原来的数组，还可以继续使用数组； 

    在这个过程中join方法不会修改我们原来的数组；

    join方法可以传递一个参数，这个参数是用来确定我们用什么东西来分隔数组中的每一项； 

    默认使用逗号分隔；也可以自定义：var a = arr.join("|");//就是用 | 分隔； 

2.*重点*：pop();//直接调用，没有参数； 

    pop 用来删除数组中的最后一项，会改变原来的数组，返回值是删除掉的元素； 

    从后删除； 

3.push();//用来给数组添加一个或多个元素(从后添加 / 追加) 

    会改变原来的数组；返回值是数组的新的长度； 

    例：var a = arr.push("a"); 

4.unshift();//从前面添加； 

    改变了数组；返回值是当前数组的长度 

    例：var a = arr.unshift("a"); 

5.shift();//从前面删除； 

    返回值是被删除的数组，会改变原数组； 

    例：var a = arr.shift(); 

6.reverse();

    把数组顺序颠倒，会改变原数组； 

    返回值为新数组；使用后新的数组等于原来的数组； 

    例：var a = arr.reverse(); 
        a = arr; 


7.sort();

    1.如果不给他传递参数，他就会按照字母的先后顺序排序 

        例：var a = arr.sort(); 

    2.按照大小进行排序： 

8.concat();

    连接两个或者是多个数组；不会改变原数组；返回一个新数组； 

    把两个数组合并成一个数组，谁调用谁再前面； 

    例：var newArr = arr.concat(arr1,arr2); 
        用arr连接arr1，arr2.。。。。； 


9.slice();[截取/获取]

    不会改变原数组；返回结果是你要的元素(把你要的元素放到一个新数组中)； 

    如果你只传一个参数，代表的是你要开始截取的位置，返回结果是从截取的位置到这个数组的末尾； 

    例：var a = arr.slice(2);//2是索引值； 

    传两个参数：第一个是代表你要截取的位置，第二个参数是你要截取到的位置；(要头不要尾)； 

    例：var b = arr.slice(2,3); 


10.splice(a,b);

    会改变原数组；如果只传一个参数，那么这个参数就是你要开始删除的位置，一直删除到末尾； 

    例： var a= arr.splice(2); 

    传两个参数：代表从第a个开始删除b个元素，返回值是删除调度元素，组成新的数组； 

    例： var a= arr.splice(2,3); 

    删除数组中的元素，并且替换成新的元素； 

    例：var a= arr.splice(2,3,"a","b","c");
