博客更新，以及本地删除文件上传博客

创建公钥和密钥

c盘 / 用户 / 7000(用户名文件夹) / .ssh文件是否存在(若存在删除)

  • 创建.ssh文件：

    • Git Bash里面填写：
    • ssh-keygen -t rsa -C “youremail@example.com”
    • .ssh文件：名字短的是密钥，名字长的是公钥；

  • 在博客里面：

    • setting — Deploy keys — Add key ：
    • title：xxx；
    • key：放入公钥里面的所有东西；
    • 勾选，添加；

本地文件夹被删除之后想要上传：

  • 新建文件夹：
    • git init
    • git clone 仓库地址
    • cd 下载的文件
    • git add -A
    • git commit -m “”
    • git push ssh的地址

更新已下载的文件：

  • git pull 文件地址
