数组的方法：重点

1.join(); 

    把数组转换成字符串；得到一个新的字符串，不会修改我们原来的数组，还可以继续使用数组； 

    在这个过程中join方法不会修改我们原来的数组；

    join方法可以传递一个参数，这个参数是用来确定我们用什么东西来分隔数组中的每一项； 

    默认使用逗号分隔；也可以自定义：var a = arr.join("|");//就是用 | 分隔； 

2.*重点*：pop();//直接调用，没有参数； 

    pop 用来删除数组中的最后一项，会改变原来的数组，返回值是删除掉的元素； 

    从后删除； 

3.push();//用来给数组添加一个或多个元素(从后添加 / 追加) 

    会改变原来的数组；返回值是数组的新的长度； 

    例：var a = arr.push("a"); 

4.unshift();//从前面添加； 

    改变了数组；返回值是当前数组的长度 

    例：var a = arr.unshift("a"); 

5.shift();//从前面删除； 

    返回值是被删除的数组，会改变原数组； 

    例：var a = arr.shift(); 

6.reverse();

    把数组顺序颠倒，会改变原数组； 

    返回值为新数组；使用后新的数组等于原来的数组； 

    例：var a = arr.reverse(); 
        a = arr; 


7.sort();

    1.如果不给他传递参数，他就会按照字母的先后顺序排序 

        例：var a = arr.sort(); 

    2.按照大小进行排序： 

8.concat();

    连接两个或者是多个数组；不会改变原数组；返回一个新数组； 

    把两个数组合并成一个数组，谁调用谁再前面； 

    例：var newArr = arr.concat(arr1,arr2); 
        用arr连接arr1，arr2.。。。。； 


9.slice();[截取/获取]

    不会改变原数组；返回结果是你要的元素(把你要的元素放到一个新数组中)； 

    如果你只传一个参数，代表的是你要开始截取的位置，返回结果是从截取的位置到这个数组的末尾； 

    例：var a = arr.slice(2);//2是索引值； 

    传两个参数：第一个是代表你要截取的位置，第二个参数是你要截取到的位置；(要头不要尾)； 

    例：var b = arr.slice(2,3); 


10.splice(a,b);

    会改变原数组；如果只传一个参数，那么这个参数就是你要开始删除的位置，一直删除到末尾； 

    例： var a= arr.splice(2); 

    传两个参数：代表从第a个开始删除b个元素，返回值是删除调度元素，组成新的数组； 

    例： var a= arr.splice(2,3); 

    删除数组中的元素，并且替换成新的元素； 

    例：var a= arr.splice(2,3,"a","b","c");

html/css

块级元素以及行内元素：

  • 块级元素：
    <div>、<p>、<h1>....<h6>、<ul>、<form>等；

    1. 每个块级元素都从新的一行开始，并且其后的元素也另起一行。（真霸道，一个块级元素独占一行）
    2. 元素的高度、宽度、行高以及顶和底边距都可设置。
    3. 元素宽度在不设置的情况下，是它本身父容器的100%（和父元素的宽度一致），除非设定一个宽度。

  • 行内元素：
    <a>、<span>、<br>、<i>、<em>、<strong>等；

    1. 和其他元素都在一行上；
    2. 元素的高度、宽度、行高及顶部和底部边距不可设置；
    3. 元素的宽度就是它包含的文字或图片的宽度，不可改变。

  • 转换：

    • 块级元素转换为行内元素：display:inline;
    • 行内元素转换为块级元素：display:block;
    • 转换为行内块：display:inline-block;

选择器：

  • 标签选择器 [element]
  • 类名选择器 .class
  • ID选择器 #id
  • 并集选择器 [element],[element]
  • 后代选择器 div p
  • 子代选择器 div>p
  • 标签指定式选择器 div.class
  • 通配符选择器 *{}

css优先级：

  • !important >行内样式 > ID选择器 > 类名选择器 > 标签选择器;

<a>标签：

  • 打开新窗口属性：target=”view_window”;
  • 防止<a>标签被点击：href=”javascript:;”;

border-style:

  • none:默认值没有边框；
  • solid:实线边框；
  • dotted:点线边框；
  • dashed:虚线边框；
  • 去除边框：border:0/none;
  • 去除轮廓线：outline-style:none;

定位：

  • 相对定位
  • 绝对定位
  • 固定定位
  • z-indext：设置元素的层级关系，只能和定位一起使用

图片的对齐方式：

  • 默认方式：vertical-aline:baseline;(基线对齐)
  • 顶端对齐：vertical-aline:top;
  • 中线对齐：vertical-aline:Middel;

  • 底部对齐：vertical-aline:Bottom;

  • 注：vertical-aline属性只适用于图片或表格；

css隐藏元素方式：

  • 超出部分隐藏：overflow:hidden;
  • 元素隐藏，不占位：display:none;
  • 元素隐藏，占位：visibility:hidden;

伪元素：

  • before:可以在元素的内容前面插入新内容;
  • after:可以在元素的内容之后插入新内容;
  • first-letter:用于向文本的首字母设置特殊样式;
  • first-line:只能用于块级元素，用于向文本的首行设置特殊样式;

<video>标签：

  • 手动播放：controls=”controls”;
  • 视频加载之后播放：autoplay=”autoplay”;
  • 循环播放：loop=”loop”;

canvas

canvas(H5标签，画布)：
H5兼容性：html5shif.js文件能够解决一些浏览器可以识别H5标签，但是canvas标签除外；
canvas只有width、height两个属性；

  1. 如果不给canvas宽高，那么他的默认宽高为300 * 150，背景为透明色；

  2. margin：0 auto；不可以使用，但是定位可以；

  3. 如果在canvas标签中直接写内容，若浏览器支持canvas这个标签则不会显示内容，若不支持则显示；

  4. canvas是通过js来进行绘制图形、制作照片等的；

  5. 步骤：

    A： 获取canvas标签 a；

    B： var ctx = a.getContex(“2d”)； //用来获取上下文(连接上下文的)，绘制一个2d图形；

    C： 绘制矩形：

    a：ctx.fillRect(x,y,width,height);       //从左向右绘制,填充矩形；
b：ctx.strokeRect();           //空心矩形；
c：ctx.clearRect(x,y,width,height);           //清空区域；

    D：路径绘制：

    a：ctx.beginPath();      //开始绘画；
b：ctx.moveTo(x,y);       //画笔的位置；
c：ctx.lineTo(x,y);        //起始位置；
    .
    .
    .
d：ctx.lineTo(x,y);         //终止位置；(必须和起始位置一至)；
e：ctx.fill();           //填充颜色(默认为黑色)；

自动闭合(没有终止位置时)：

    a：ctx.moveTo(x,y);
    b：ctx.lineTo(x,y);
        .
        .
        .
    c：ctx.closePath();       //自动闭合；
    d：ctx.stroke();

E：绘制圆/圆弧：

    a：ctx.beginPath();
    b：ctx.arc(x,y,r,start,end,方向)；//x，y:圆心  /  r：半径   
                        start：开始位置，end：结束位置(以x轴为基准)      
                                 方向：true逆时针，false顺时针(默认false)；
    c：ctx.fill();
  1. 填充颜色：ctx.fillStyle=”#aaa”; //在开始画之前进行填充；
  1. 贝塞尔曲线(二次)：
    ctx.quadraticCurveTo(x,y,x1,y1);    //x,y:控制点    /   x1,y1: 终止点；

canvas用法

canvas

<canvas> 是 HTML5 新增的元素，可用于通过使用JavaScript中的脚本来绘制图形。例如，它可以用于绘制图形，制作照片，创建动画，甚至可以进行实时视频处理或渲染。

基本用法：

  • 创建canvas html上下文对象

    • canvas对象只有两个属性,width和height,并且者两个属性都是可选属性,当不设置的话默认是:300*150的大小

  • canvas 标签起初是空白的我们要向渲染(绘制),就得通过脚本(js)进行获取,通过 .getContext的方法获取上下文

画一个以（x,y）为圆心的以radius为半径的圆弧（圆），从startAngle开始到endAngle结束，按照anticlockwise给定的方向（默认为顺时针）来生成。

  • 绘制贝塞尔曲线:quadraticCurveTo()

    quadraticCurveTo(cp1x, cp1y, x, y)
    绘制二次贝塞尔曲线，cp1x,cp1y为一个控制点，x,y为结束点。

添加样式和颜色

  • 添加填充色: fillStyle
  • 添加轮廓色: strokeStyle

    线性:

  • lineWidth = value

    设置线条宽度。

  • lineCap = type

    设置线条末端样式。

  • lineJoin = type

    设定线条与线条间接合处的样式。

  • miterLimit = value

    限制当两条线相交时交接处最大长度；所谓交接处长度（斜接长度）是指线条交接处内角顶点到外角顶点的长度。

  • getLineDash()

    返回一个包含当前虚线样式，长度为非负偶数的数组。

  • setLineDash(segments)

    设置当前虚线样式。

  • lineDashOffset = value

    设置虚线样式的起始偏移量。

  • app.js文件是属于全局的js文件。

  1. 一个页面是由四个文件组成的：

    • wxml文件：页面的结构 —– 相当于html文件
    • wxss文件：页面的样式 —– 相当于css文件
    • js文件：行为 / 逻辑
    • json文件：页面的配置信息

  2. 每一个页面需要单独写一个文件夹，放到pages文件夹中，并且这个页面的的文件命名都是有要求的：

    • 假如你要创建一个index文件，那么首先要在pages文件里面创建一个index文件夹，并且这个文件夹下的文件都要叫index
      例如：index.wxml、index.wxss……..

  3. 添加一个页面：

    • 需要创建这个页面的文件夹；
    • 配置这个页面到小程序中：
      • 在我们项目中的app.json文件中找到pages的字段，给这个字段添加内容，添加你要在小程序中添加的页面。
  1. app.json：是相当于小程序的全局配置，包括了小程序的所有页面路径、界面表现、网络超时时间、底部tab等。
