canvas用法

canvas

<canvas> 是 HTML5 新增的元素，可用于通过使用JavaScript中的脚本来绘制图形。例如，它可以用于绘制图形，制作照片，创建动画，甚至可以进行实时视频处理或渲染。

基本用法：

  • 创建canvas html上下文对象

    • canvas对象只有两个属性,width和height,并且者两个属性都是可选属性,当不设置的话默认是:300*150的大小

  • canvas 标签起初是空白的我们要向渲染(绘制),就得通过脚本(js)进行获取,通过 .getContext的方法获取上下文

画一个以（x,y）为圆心的以radius为半径的圆弧（圆），从startAngle开始到endAngle结束，按照anticlockwise给定的方向（默认为顺时针）来生成。

  • 绘制贝塞尔曲线:quadraticCurveTo()

    quadraticCurveTo(cp1x, cp1y, x, y)
    绘制二次贝塞尔曲线，cp1x,cp1y为一个控制点，x,y为结束点。

添加样式和颜色

  • 添加填充色: fillStyle
  • 添加轮廓色: strokeStyle

    线性:

  • lineWidth = value

    设置线条宽度。

  • lineCap = type

    设置线条末端样式。

  • lineJoin = type

    设定线条与线条间接合处的样式。

  • miterLimit = value

    限制当两条线相交时交接处最大长度；所谓交接处长度（斜接长度）是指线条交接处内角顶点到外角顶点的长度。

  • getLineDash()

    返回一个包含当前虚线样式，长度为非负偶数的数组。

  • setLineDash(segments)

    设置当前虚线样式。

  • lineDashOffset = value

    设置虚线样式的起始偏移量。

微信小程序

  • app.js文件是属于全局的js文件。

  1. 一个页面是由四个文件组成的：

    • wxml文件：页面的结构 —– 相当于html文件
    • wxss文件：页面的样式 —– 相当于css文件
    • js文件：行为 / 逻辑
    • json文件：页面的配置信息

  2. 每一个页面需要单独写一个文件夹，放到pages文件夹中，并且这个页面的的文件命名都是有要求的：

    • 假如你要创建一个index文件，那么首先要在pages文件里面创建一个index文件夹，并且这个文件夹下的文件都要叫index
      例如：index.wxml、index.wxss……..

  3. 添加一个页面：

    • 需要创建这个页面的文件夹；
    • 配置这个页面到小程序中：
      • 在我们项目中的app.json文件中找到pages的字段，给这个字段添加内容，添加你要在小程序中添加的页面。
  1. app.json：是相当于小程序的全局配置，包括了小程序的所有页面路径、界面表现、网络超时时间、底部tab等。
